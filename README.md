# **A**ll-purpose **Sch**eduler **ASch**

This is a simple event-driven single-thread scheduler for embedded devices. Designed mainly for learning embedded C++.

### Prerequisites

- MinGW + GCC for C++
- Python 3.7
- SCons 3.0.1

### Building

Run SCons on the root of the project.
